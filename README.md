How to use
----------
Clone the repository:
```shell
$ git clone git@github.com:scully87/yelp_clone.git
```

Change into the directory:
```shell
$ cd yelp_clone
```

Open from command line:
```shell
$ rackup
....then open in browser: http://localhost:3000
```
